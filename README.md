# AI Daily Intel Hub

AI Daily Intel Hub is a modern daily intelligence briefing application with a clean, beautiful user interface that showcases the latest global news and in-depth analysis. Each daily briefing traces back to the original news sources and includes links below each news item. Users can listen to the briefings through the built-in web player.

Official website: https://www.dailyintel.qzz.io/

## Getting Started

### Local Development

**Prerequisites**: Node.js 18+ 

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Open in browser:
   ```
   http://localhost:3000
   ```

### Build for Production

```bash
npm run build
```

Build output will be generated in the `dist` directory.
